# BridgeCare - Homeless Support System

A comprehensive platform connecting NGOs, healthcare providers, hotels, volunteers, and donors to support homeless individuals.

## Features

- Multi-role authentication system
- Real-time data tracking
- Dashboard for each stakeholder
- Supabase integration for data storage

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up environment variables in `.env` file
4. Run the application: `python app.py`

## Roles

- Admin: Overall system management
- Healthcare: Medical assistance tracking
- Hotel: Food donation management
- NGO: Resource coordination
- Volunteer: Homeless individual registration
- Donor: Contribution management
